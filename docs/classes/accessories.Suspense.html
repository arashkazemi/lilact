    Class Suspense

    Suspense - boundary for asynchronous loading. Shows a fallback UI while descendant thrown promises are pending, and renders real content once resolved.

    Component

    Suspense

    Param: props

    Param: props.fallback

    Element shown while descendants are loading.

    Param: props.children

    Suspended children.

    Param: props.minDelay

    A delay before showing the fallback to prevent flicker.

    Param: props.minShowTime

    A minimum time of fallback visibility to prevent flickers.
